# Woya
Application de mise en relation de vente de service et d'achat

## Prérequis

- Node.js >= 20.19 (recommandé : dernière version LTS)
- npm >= 10

> **Remarque :** L'utilisation d'une version de Node.js antérieure à 20.19 entraîne des erreurs `EBADENGINE` lors de l'installation des dépendances et empêche l'exécution des commandes Angular CLI telles que `ng new`.
